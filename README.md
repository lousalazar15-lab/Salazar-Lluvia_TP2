# TP2_Lluvia

Thème: un petit restaurant
Tes un manager qui doit manager ses employées qui sont entrain de faire n'importe quoi dans 2 pièces différentes,
sauf que l'employé de la premiere pièce a bloqué la porte donc vous êtes enfermé dedans jusqu'à ce que tu trouve comment partir de cette pièce avant que les employées de la deuxième pièce la brule.
